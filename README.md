# Instant

## Kompilacja
Program kompilujemy przy użyciu polecenia `make`

## Użyte biblioteki
- containers
- mtl
- text
- filepath
- process
- text-format

## Struktura
- src/main - pliki main dla kompilatora llvm i jvm
- src/instant - właściwa implementacja kompilatorów
- src/third/ - pliki parsera
